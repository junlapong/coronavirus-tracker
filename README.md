# COVID-19 TRACKER

Example from [Building a Coronavirus tracker app with Spring Boot and Java - Java Brains Tutorial](https://youtu.be/8hjNG9GZGnQ)

## Prerequisite

- Java 11+
- Maven

## Run

```
mvn spring-boot:run
```

## Open

- http://localhost:8080
